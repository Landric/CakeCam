CakeCam Documentation

This is CakeCam. CakeCam is a handy tool for sharing Cake! and left over food with the lab. Follow the simple instructions and you too can share cake.

  1. Place the Cake! on the large cross on the table below the camera
  2. Press the "Cake" button on the keyboard (<F1> key)
  3. An email with the [CakeCam] in the subject line is sent to the b32-all mailing list.

About CakeCam

CakeCam was created by Patrick McSweeney from University of Southampton Faculty of Physical and Applied Science. The source code is open source and available from https://github.com/patrickmcsweeney/CakeCam. The files for CakeCam can be found in /Users/or08/CakeCam. The project is written in Java using the OpenImaj (http://www.openimaj.org/) library developed at the University of Southampton.
